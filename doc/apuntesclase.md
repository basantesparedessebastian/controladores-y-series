## Tipo de datos en java

 En java hay 3 tipos de datos

 1. Tipos primitivos
 2. Tipos de referencia (clases)
 3. Void (caso especial)

### Tipos primitivos (los mas basicos)

son los tipos mas simples que existen .guardan el valor directamente.no necesitan declaraciones son mucho mas rapidos.

1. *int*= numeros enteros
2. *double*= numeros decimales
3. *float*= decimales (menos precisos)
4. *boolean*=verdadero o falso
5. *char* = una sola letra
6. *byte*= numeros pequeñops como archivos o datos de red
7. *short*= enteros pequeños (se usa rara vez)
8. *long*= enteros muy grandes (numeros demasiado grnades con mas de 10 cifras)

### Tipos no primitivos (por referencia)

se usan para los valores que no van a cambiar
*String*= es una **clase** que maneja texto/cadenas
*void* = es un caso especial significa vacio o nada.Solo se usa como tipo de retorno de metodos que no deveulven nada

### ¿cuando se utilizan cada uno?

| si necesitas | Usa | ejemplo |
|--------------|-----|---------|
| un numero entero | int | int contador=0 |
| un numero decimal | double | double iva =0,21 |
| un SI/NO , verdadero/falso | boolean encendido = true |
| una sola letra | char | char genero = "F" |
| un texto (nombre, mensje , etc) | String | string nombre= "ana" |
| un metodo que no devuelve nada | void | public void imprimir() { } |

### ¿donde se utilizan?

| lugar en el codigo | que tipos se pueden usar |
|--------------------|--------------------------|
| variables | int, double, boolean, String, etc |
|Parametros de metodos | int, double, String,etc |
|Retorno de metodos | int, double , String, ....o void si no retorna nada |
|Atributos de clase | int , double ,string, etc |
